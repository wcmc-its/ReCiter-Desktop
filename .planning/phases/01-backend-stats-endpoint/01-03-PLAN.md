**File 1: Create `api/routers/stats.py`** (per D-07)

This is a thin HTTP layer. All computation is in `stats_service.py`. The router:
1. Creates an `APIRouter` with `prefix="/api/stats"` and `tags=["stats"]`
2. Defines one `GET ""` endpoint that injects `db: Session = Depends(get_db)` and calls `compute_stats(db)`
3. Returns the result dict directly (routers return plain dicts, not Pydantic models -- per existing pattern)

Exact file contents:
```python
from fastapi import APIRouter, Depends
from sqlalchemy.orm import Session

from api.database import get_db
from api.services.stats_service import compute_stats

router = APIRouter(prefix="/api/stats", tags=["stats"])


@router.get("")
def get_stats(db: Session = Depends(get_db)):
    return compute_stats(db)
```
