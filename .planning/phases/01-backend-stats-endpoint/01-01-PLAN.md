**Test function signatures and assertions (exact names from VALIDATION.md):**

1. `test_roc_output()` -- STATS-01: Call `compute_roc(SCORES_01, LABELS)`. Assert result is a dict with keys `fpr`, `tpr`, `auc`, `ci_lower`, `ci_upper`, `ci_degraded`. Assert `fpr` and `tpr` are lists. Assert `0 < result["auc"] <= 1.0`. Assert `ci_lower <= auc <= ci_upper`.

4. `test_calibration_bins()` -- STATS-02: Call `compute_calibration(SCORES_01, LABELS)`. Assert result is a list of length exactly 10. 5. `test_calibration_n_per_bin()` -- STATS-02: Call `compute_calibration(SCORES_01, LABELS)`. Assert `sum(b["n"] for b in result) == len(SCORES_01)` (n-per-bin sums to total n).

Assert all items have keys: `person_id`, `pmid`, `score`, `assertion`, `disagreement`.
