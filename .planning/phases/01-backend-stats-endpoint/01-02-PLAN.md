**Function 1: `check_viability(n, assertions)`**
- Signature: `def check_viability(n: int, assertions: list) -> tuple[bool, dict]:`
- Returns `(is_blocked, response_dict)` where `is_blocked=True` means caller should return response_dict immediately
- Logic (per D-06, STATS-06):
  - If `n == 0`: return `(True, {"viable": False, "below_n_threshold": True, "error": "no_data", "message": "No joined (score, assertion) pairs found. Run the pipeline and import curations first."})`
  - If `len(set(assertions)) < 2`: return `(True, {"viable": False, "below_n_threshold": n < 50, "error": "single_class", "message": f"All assertions are the same label: {list(set(assertions))[0]}. Cannot compute ROC or PR curves."})`
  - Otherwise: return `(False, {"viable": n >= 50, "below_n_threshold": n < 50})`

**Function 2: `prepare_joined_data(rows)`**
- Signature: `def prepare_joined_data(rows) -> tuple[np.ndarray, np.ndarray, np.ndarray, list, list, list, list]:`
- `rows` is the result of the SQLAlchemy JOIN query (list of tuples with `.PersonArticleScore` and `.Curation` attributes)
- Per D-01: For each person_id, if any row has `model_type == "feedbackIdentity"`, use only feedbackIdentity rows for that person; otherwise use identityOnly rows
- Per D-03: Multiply `calibrated_score * 100` to get 0-100 scale. Guard with `float(row.PersonArticleScore.calibrated_score or 0)`
- Returns: `(scores_100, scores_01, labels_binary, assertions, selected_rows, person_ids, pmids)` where:
  - `scores_100` = numpy array on 0-100 scale
  - `scores_01` = `scores_100 / 100` for sklearn APIs
  - `labels_binary` = numpy array of 1 (ACCEPTED) / 0 (REJECTED)
  - `assertions` = list of "ACCEPTED" / "REJECTED" strings
  - `selected_rows` = the filtered row list (for disagreements)
  - `person_ids` = list of person_id strings (parallel to scores)
  - `pmids` = list of pmid strings (parallel to scores)

**Function 3: `compute_roc(scores_01, labels_binary, n_resamples=1000)`**
- Signature: `def compute_roc(scores_01: np.ndarray, labels_binary: np.ndarray, n_resamples: int = 1000) -> dict:`
- Per STATS-01, D-04, D-05:
  - `fpr, tpr, _ = roc_curve(labels_binary, scores_01)`
  - `auc_val = roc_auc_score(labels_binary, scores_01)`
  - Bootstrap CI: `rng = np.random.default_rng()`; loop `n_resamples` times, sample `idx = rng.integers(0, n, size=n)`, skip if `len(np.unique(labels_binary[idx])) < 2`, append `roc_auc_score(labels_binary[idx], scores_01[idx])`
  - Time the bootstrap with `time.perf_counter()`; set `ci_degraded = elapsed > 2.0` (D-05)
  - CI = `np.percentile(boot_aucs, [2.5, 97.5])` if boot_aucs else `[0.0, 0.0]`
  - Return dict with all numpy types converted: `{"fpr": fpr.tolist(), "tpr": tpr.tolist(), "auc": float(auc_val), "ci_lower": float(ci[0]), "ci_upper": float(ci[1]), "ci_degraded": bool(ci_degraded)}`

**Function 4: `compute_calibration(scores_01, labels_binary)`**
- Signature: `def compute_calibration(scores_01: np.ndarray, labels_binary: np.ndarray) -> list[dict]:`
- Per STATS-02: 10 uniform bins on 0-1 scale (mapped to 0-100 labels)
- Do NOT rely on `calibration_curve` alone -- it drops empty bins. Instead:
  - `cal_edges = np.linspace(0, 1, 11)` (11 edges = 10 bins)
  - `bin_idx = np.clip(np.digitize(scores_01, cal_edges) - 1, 0, 9)`
  - For each bin b in 0-9: compute `mask = bin_idx == b`, then `n = int(mask.sum())`, `mean_score = float(scores_01[mask].mean() * 100) if n > 0 else None`, `fraction_positive = float(labels_binary[mask].mean()) if n > 0 else None`
  - Bucket label: `f"{b*10}-{(b+1)*10}"`
  - Return list of 10 dicts: `[{"bucket": label, "mean_score": mean_score, "fraction_positive": frac_pos, "n": n}, ...]`

**Function 5: `compute_pr(scores_01, labels_binary)`**
- Signature: `def compute_pr(scores_01: np.ndarray, labels_binary: np.ndarray) -> dict:`
- Per STATS-03: `prec, rec, _ = precision_recall_curve(labels_binary, scores_01)`
- `auc_pr = average_precision_score(labels_binary, scores_01)`
- `pr_baseline = float(labels_binary.mean())` -- actual positive rate, NOT 0.5
- Return: `{"precision": prec.tolist(), "recall": rec.tolist(), "auc_pr": float(auc_pr), "pr_baseline": pr_baseline}`

**Function 6: `compute_distribution(scores_100, assertions)`**
- Signature: `def compute_distribution(scores_100: np.ndarray, assertions: list) -> list[dict]:`
- Per STATS-04: 10-point buckets covering 0-100
- `dist_edges = np.arange(0, 101, 10)` (11 edges = 10 buckets)
- `bucket_idx = np.clip(np.digitize(scores_100, dist_edges) - 1, 0, 9)`
- `accepted_mask = np.array([a == "ACCEPTED" for a in assertions])`
- `rejected_mask = ~accepted_mask`
- For each bucket b in 0-9: `mask = bucket_idx == b`, `accepted = int((mask & accepted_mask).sum())`, `rejected = int((mask & rejected_mask).sum())`
- Return list of 10 dicts: `[{"bucket": f"{b*10}-{(b+1)*10}", "accepted": accepted, "rejected": rejected}, ...]`

**Function 7: `compute_disagreements(scores_100, assertions, person_ids, pmids, names=None)`**
- Signature: `def compute_disagreements(scores_100: np.ndarray, assertions: list, person_ids: list, pmids: list, names: dict | None = None) -> list[dict]:`
- Per STATS-05: `assertion_values = np.array([100.0 if a == "ACCEPTED" else 0.0 for a in assertions])`
- `disagreements = np.abs(scores_100 - assertion_values)`
- `top_indices = np.argsort(disagreements)[::-1][:10]`
- `names` is an optional dict of `{person_id: {"first_name": str, "last_name": str}}` -- if None, names are omitted from results
- Return list of dicts (up to 10): `[{"person_id": person_ids[i], "pmid": pmids[i], "score": float(scores_100[i]), "assertion": assertions[i], "disagreement": float(disagreements[i]), "first_name": names.get(pid, {}).get("first_name") if names else None, "last_name": names.get(pid, {}).get("last_name") if names else None}, ...]`

**Function 8: `compute_stats(db)`**
- Signature: `def compute_stats(db: Session) -> dict:`
- This is the orchestrator called by the router. It:
  1. Queries DB: `rows = db.query(PersonArticleScore, Curation).join(Curation, (PersonArticleScore.person_id == Curation.person_id) & (PersonArticleScore.pmid == Curation.pmid)).all()`
  2. Calls `prepare_joined_data(rows)` to get `scores_100, scores_01, labels_binary, assertions, selected_rows, person_ids, pmids`
  3. Calls `check_viability(len(scores_01), assertions)` -- if `is_blocked`, return the error dict immediately
  4. Computes all 5 stats: `roc = compute_roc(scores_01, labels_binary)`, `calibration = compute_calibration(scores_01, labels_binary)`, `pr = compute_pr(scores_01, labels_binary)`, `distribution = compute_distribution(scores_100, assertions)`, disagreements (with Identity lookup for names)
  5. For disagreements: query `Identity` table for the person_ids in the top-10, build names dict
  6. Merges viability flags with results: `{**viability_flags, "n": len(scores_01), "roc": roc, "calibration": calibration, "calibration_viable": len(scores_01) >= 50, "pr": pr, "distribution": distribution, "disagreements": disagreements}`
